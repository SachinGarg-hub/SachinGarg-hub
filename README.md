## 🚀 Featured Projects

<details>
<summary><b>🔍 URLScope — ML-Powered Phishing URL Detection</b></summary>
<br/>

Real-time phishing URL detection engine combining an ML classifier with domain-reputation intelligence and SHAP-based explainability, engineered to eliminate false positives at scale.

| Attribute | Detail |
|---|---|
| **Stack** | Python, Scikit-learn, SHAP, Pandas |
| **Scale** | Trained on augmented multi-domain URL dataset (global + Indian domains) |
| **Performance** | Resolved critical scheme-presence feature leakage & root-domain bias |
| **Security** | Application-layer domain-reputation allowlist keyed on registrable domain |
| **Impact** | Eliminated systemic false-positive bias in safe-class predictions |
| **Repository** | [SachinGarg-hub/URLScope](https://github.com/SachinGarg-hub/URLScope) |

Diagnosed and fixed a critical training-data gap where safe-class URLs lacked bare root-domain representation, confirmed via SHAP diagnostics. Rebuilt the feature pipeline around canonical URL forms, removed leaking features like `has_https`, and shipped dedicated diagnostic tooling (`sanity_check.py`, `diagnose_dataset.py`, `explain_prediction.py`) for ongoing model auditing.

</details>

| **Performance** | SSRF-protected scanning layer with deterministic scoring engine |
| **Security** | Hardened backend against scanner-abuse and SSRF vectors |
| **Impact** | Automated professional-grade PDF security reports end-to-end |
| **Repository** | Available on request |

Built a FastAPI backend performing SSRF-protected Nmap scans, paired with a deterministic rule-based risk-scoring engine and Claude API integration for human-readable narrative analysis. Findings are compiled into exportable PDF reports via ReportLab, presented through a Vite/React frontend styled as a security operations console.

</details>
